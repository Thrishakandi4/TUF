# Interactive Calendar Component

This is a project for a frontend engineering challenge, built with React. It's an interactive calendar component that allows users to select a date range and take notes for specific dates.

## Features

*   **Interactive Calendar:** Navigate through months with a flipping animation.
*   **Date Range Selection:** Click to select a start and end date. A third click clears the selection.
*   **Notes:** Add notes to a selected date. Notes are saved in local storage.
*   **Responsive Design:** The layout adapts to both desktop and mobile screens.

## Tech Stack

*   **React**
*   **date-fns:** For date manipulation and formatting.
*   **CSS:** For all styling and animations.

## How to Run

1.  **Clone the repository.**
2.  Navigate into the `frontend` directory:
    ```bash
    cd frontend
    ```
3.  Install the necessary packages:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm start
    ```
    The application will be available at `http://localhost:3000`.
